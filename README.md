# HttpSpy
A powerful and highly efficient network debugging tool for Roblox (and exploits)

## Usage
```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/NotDSF/HttpSpy/main/init.lua"))();
```

## Features
- Request Reconstructing
- Syntax Highlighting
- WebSocket support
- Lightweight
- Auto JSON Decoding
- Easy to use
